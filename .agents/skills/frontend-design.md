# Frontend Design Skill

## Purpose
Provide expert guidance on creating distinctive, modern web interfaces that avoid generic "AI slop" aesthetics.

## Core Principles

### 1. Bold Aesthetic Directions
- Choose a clear visual identity (not "safe" defaults)
- Commit to specific design decisions
- Create visual tension through contrast

### 2. Typography as a Design Element
- Use font personalities intentionally
- Create dramatic size contrasts
- Play with weights and styles
- Consider custom fonts for distinctiveness

### 3. Color with Purpose
- Limited palette (2-3 primary colors)
- Use color for hierarchy, not decoration
- Ensure accessibility (4.5:1 contrast minimum)
- Theme support (light/dark modes)

### 4. Spatial Composition
- Generous white space
- Intentional proximity relationships
- Break grid expectations occasionally
- Consider depth layers (z-axis)

### 5. Motion Design
- Purposeful animations (not decorative)
- Respect user preferences (reduced motion)
- 200-500ms duration sweet spot
- Ease-in-out for natural feel

## Application to Academic Portfolios

### What Makes Academic Portfolios Distinctive

1. **Authentic Voice**
   - Show personality while maintaining professionalism
   - Use active voice in descriptions
   - Highlight unique research contributions

2. **Visual Hierarchy**
   - Name/title as primary focal point
   - Research highlights with clear metrics
   - Publications organized by impact

3. **Trust Signals**
   - Institutional affiliations
   - Peer recognition (awards, citations)
   - Professional photography

4. **Accessibility First**
   - Academic content must be universally accessible
   - Print-friendly stylesheets
   - High contrast options

## Avoid These "AI Slop" Patterns

- Generic gradient backgrounds without purpose
- Overused "tech blue" as primary color
- Decorative animations that distract from content
- Card layouts with no visual hierarchy
- Stock photography instead of authentic images
- Excessive rounded corners on everything
- Cookie-cutter "modern" designs without personality

## Implementation Checklist

- [ ] Design has a clear visual identity
- [ ] Typography creates clear hierarchy
- [ ] Color choices support accessibility
- [ ] Animations serve functional purposes
- [ ] Layout breaks from generic templates
- [ ] Images are authentic (not stock)
- [ ] Mobile experience is first-class
- [ ] Print stylesheet is functional
