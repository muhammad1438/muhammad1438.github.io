## Implementation Notes

### Image Optimization Guidelines

1. **Format Selection**:
   - Icons: SVG (preferred) or PNG with transparency
   - Photos: WebP with JPEG fallback
   - Patterns: SVG or optimized PNG (tileable)

2. **Size Guidelines**:
   - Hero images: 1920x1080 minimum, compress to <500KB
   - Icons: 64x64 or 128x128 for retina displays
   - Thumbnails: 400x300 or 400x400
   - Full-width banners: 1920x400-600

3. **Lazy Loading**:
   - Add `loading="lazy"` to all images below the fold
   - Use `eager` loading only for hero/above-fold images

4. **Alt Text**:
   - All images must have descriptive alt text
   - Decorative images can use `alt=""` with `role="presentation"`

### AI Generation Best Practices

- Use `--style raw` for more photorealistic results
- Use `--q 2` for highest quality output
- Generate multiple variations and select the best
- For Midjourney, use `--v 6` for latest version
- For consistent colors, reference hex codes in prompts
